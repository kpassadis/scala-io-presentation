# ZIO for Data Scientists

A presentation exploring how to build robust, functional data pipelines and machine learning workflows using Scala and ZIO—without the Category Theory jargon.

👉 **[View the Live Presentation Here](https://kpassadis.github.io/scala-io-presentation/)**

## Overview
This deck covers:
* The realities of the Data Science workflow.
* Replacing heterogeneous tools (Ray, Airflow, FastAPI) with a unified ZIO pipeline.
* Practical examples using ZIO Streams, ZIO Parser, and ZIO JSON.
* Dependency Injection for ML models using ZLayer.
